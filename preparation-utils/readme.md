# 项目含义

机会是留给有准备的人的，这个项目的目标就是做好准备，应对问题。

# 项目内容

## utils

工具类，目前主要有使用的有

- batch 批处理类
- io 文本文件处理类
- utils 配置、数据库、数字等杂项工具

# 版本历史

## 0.5.0

没有记错的话应该是使用 SVN 管理的最后一个版本，后续放到了 github 上面。

## 0.7.0

意识到批处理还不够完善，很多错别字，这个版本就把这些问题修正，然后准备在下一个版本升级以下批处理处理的，使其更加好用。
